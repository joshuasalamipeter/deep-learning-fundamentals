# Deep Learning Fundamentals Portfolio

This repository contains 12 end-to-end deep learning projects completed as part of the WorldQuant University Deep Learning Fundamentals Lab 1 & 2.

The projects progress from foundational concepts such as tensor operations and gradient descent to advanced architectures including Convolutional Neural Networks (CNNs), Recurrent Neural Networks (RNNs), and Generative Adversarial Networks (GANs).

---

## 🚀 Projects Overview

### 🔹 Regression & Neural Network Foundations
DL_Lab_1. **Concrete Strength Prediction**
   - Built regression models from linear to deep neural networks
   - Explored non-linearity and model complexity

DL_Lab_2. **Heart Disease Classification**
   - Implemented perceptron and neural networks
   - Manual forward and backpropagation

---

### 🔹 Advanced Classification
DL_Lab_3. **Protein Localization (Multiclass Classification)**
   - Handled 10-class classification problem
   - Addressed class imbalance and generalization

DL_Lab_4. **Image Classification with Deep Networks**
   - Trained deep MLP architectures on CIFAR-10
   - Studied optimization challenges in deep networks

---

### 🔹 Computer Vision with CNNs
DL_Lab_5. **Pet Breed Classification**
   - Built CNN models for image classification
   - Compared CNNs with fully connected networks

DL_Lab_6. **Seedling Classification (Transfer Learning)**
   - Applied pre-trained models
   - Used data augmentation and fine-tuning

---

### 🔹 Sequence Modeling & Generative Models
DL_Lab_7. **RNN for Time Series Prediction**
   - Modeled sequential data
   - Explored vanishing gradient problem

DL_Lab_8. **LSTM & GRU Networks**
   - Implemented gating mechanisms
   - Improved sequence learning performance

---

### 🔹 Representation Learning & Generation
DL_Lab_9. **Autoencoders & VAEs**
   - Learned latent representations
   - Applied anomaly detection and dimensionality reduction

DL_Lab_10. **GANs for Image Generation**
   - Built generator and discriminator networks
   - Addressed training instability

---

### 🔹 Advanced Topics
DL_Lab_11. **Transfer Learning & Fine-Tuning**
   - Adapted pre-trained models
   - Improved performance on new datasets

DL_Lab_12. **AI Ethics & Responsible AI**
   - Explored bias, fairness, and transparency
   - Evaluated ethical implications of AI systems
